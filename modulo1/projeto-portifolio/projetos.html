VENDA DE INGRESSO

Este projeto consiste em coletar dados do usuário para venda de ingresso, para semifinais, decisão do terceiro lugar e final do jogo, onde, ao término do preenchimento, o programa finaliza a coleta de dados e retorna os dados completos da compra, com: Nome do cliente, tipo de jogo (Semifinais, Decisão do terceiro lugar ou Final).

Tecnologia Ultilizada

  1. JavaScript
    • Utilizando condicionais, como if e else
    • Objetos e Arrays
    • Variáveis
    • Operadores e Lógica de Programação
